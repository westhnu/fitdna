█ 분석 개요
----------------------------------------------------------------------------------------------------
  • 분석 대상: FIT-DNA 전처리 데이터 (국민체력100 기반)
  • 총 데이터 건수: 115,983건
  • FIT-DNA 유형: 8가지 (Power/Light × Flexibility/Stiff × Endurance/Quick)
  • 분석 기간: 체력 측정 데이터 기준
  • 주요 지표: 근력(strength_z), 유연성(flex_z), 지구력(endurance_z)

█ 4. 핵심 인사이트 및 제안
----------------------------------------------------------------------------------------------------

  [4-1] 유형 분포 인사이트
    • 가장 많은 유형: PFE (21.30%)
      → 서비스 타겟층으로 우선 고려 필요
    • 가장 적은 유형: PSQ (8.13%)
      → 희소 유형, 특화 콘텐츠 제공 검토

  [4-2] 운동 매칭 알고리즘 제안
    • 유사도 레벨 1 (매우 유사): 거리 < 1.2
      예) LSE ↔ PSE (거리 1.075)
    • 유사도 레벨 2 (유사): 거리 1.2 ~ 2.0
    • 유사도 레벨 3 (다름): 거리 > 2.0
      예) LSQ ↔ PFE (거리 2.889) - 매칭 비추천

  [4-3] 유형별 운동 추천 방향
    • PFE:
      강점: 전반적 높은 체력 | 추천: 고강도 운동, 크로스핏, HIIT
    • LSQ:
      약점: 근력·지구력 낮음 | 추천: 저강도 유산소, 가벼운 근력 운동
    • PSE:
      약점: 유연성 낮음 | 추천: 스트레칭, 요가, 필라테스 우선
    • LFQ:
      약점: 근력·지구력 낮음 | 추천: 저강도 유산소, 가벼운 근력 운동

  [4-4] 연령대별 맞춤 전략
    • 청소년: 주요 유형 PFE
      → 성장기 체력 향상 프로그램 강화
    • 성인: 주요 유형 PFE
      → 직장인 대상 짧은 시간 고효율 운동


█ 5. Phase 2 준비 사항
----------------------------------------------------------------------------------------------------
  [5-1] 운동 메이트 매칭 알고리즘 개발
    • 유사도 계산 함수 구현 (유클리드 거리, 코사인 유사도)
    • 필터링 옵션: 운동 선호도, 시간대, 위치
    • 매칭 스코어링 시스템 구축

  [5-2] 부상 위험도 예측 모델
    • 입력: 컨디션(통증/피로/긴장) + FIT-DNA 유형
    • 출력: 부위별 위험도 (허리/무릎/어깨)
    • 모델: 로지스틱 회귀 또는 랜덤 포레스트

  [5-3] 운동 추천 규칙 엔진
    • 유형별 강점/약점 기반 운동 매핑
    • 난이도별 루틴 (초보/중급/고급)
    • 실내/실외 옵션 구분

  [5-4] 체력 변화 예측 모델
    • 재측정 데이터 활용 시계열 분석
    • 6개월 후 체력 지표 예측
    • Consistency Score 산출
