## 📊 핵심 수치

```
총 데이터: 115,983건
FIT-DNA 유형: 8가지
가장 많은 유형: PFE (21.3%) - 전천후 체력왕
가장 적은 유형: PSQ (8.1%) - 순간 파워형
최소 필수 입력: 5개 측정 항목
```

---

## 🎯 주요 인사이트

1. **PFE 유형이 가장 많음 (21.3%)**
   → 균형잡힌 체력을 가진 사람이 많음

2. **성별 차이가 큼**
   - 악력: 남성이 여성보다 1.7배 높음
   - 유연성: 여성이 남성보다 20% 높음
   → **성별 구분 필수!**

3. **매칭 가능성**
   - 레벨 1 (매우 유사): 4쌍
   - 예: LSE ↔ PSE (거리 1.075)
